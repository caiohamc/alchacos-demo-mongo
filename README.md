# MongoDB Demo

## Endpoints
* MONGO
  * POST    /api/v1/foos/traditional (**recebe o request abaixo**)
  * GET     /api/v1/foos/traditional
* REACTIVE MONGO
  * POST    /api/v1/foos/reactive (**recebe o request abaixo**)
  * GET     /api/v1/foos/reactive
